# Student Management System

A web-based Student Management System developed using Python Flask and SQLite.

## Features

* User Login and Signup
* Add Student Records
* View Student Records
* Update Student Information
* Delete Student Records
* Secure Authentication System

## Technologies Used

* Python
* Flask
* SQLite
* HTML
* CSS
